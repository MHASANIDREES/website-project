рџ“ќ Project Overview

UOL Tech Inventory Management System is a fully interactive web application that allows users to manage tech products efficiently. The project demonstrates CRUD operations, search/filter functionality, and dark/light theme toggling using JavaScript ES6 and Tailwind CSS.

This application is designed to help students learn how to manipulate arrays, objects, and the DOM dynamically, while building a modern, responsive UI.

Dark/Light Mode: Toggle between themes with localStorage persistence.
Responsive Design: Works seamlessly on desktop and mobile devices.
Default Product Data: Preloaded products to test editing, deleting, and updating.
