About Us

This project is a Tech Inventory Management System developed for UOL.

It demonstrates real-world JavaScript concepts including:

  • CRUD Operations using Array of Objects
  • Search, Filter & Sorting Logic
  • Advanced Array Methods (map, filter, reduce)
  • DOM Manipulation
  • Dark/Light Mode using localStorage

The system is designed with a modular structure using ES6 and Tailwind CSS for modern UI design.
